# Blog insights

Wordpress.com と連携してブログアクセス、キーワードアクセス数を分析するツール。

URL -> https://blog-insights.vercel.app/



## 機能

- ブログアクセス数、記事アクセス数、キーワードアクセス数一覧ページ(トップページ)
- キーワードアクセス数一覧ページ

## 今後増やす予定のページ

- 共起キーワードのアクセス数
- 記事アクセス数一覧ページ
